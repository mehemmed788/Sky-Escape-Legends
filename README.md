# Sky-Escape-Legends
Addictive 2D arcade game with progression system, shop, and 3D-style visuals. Built with Python + Pygame.
A modern Flappy Bird-style arcade game made with Python and Pygame.

## Features

* Smooth 2D gameplay with pseudo-3D visual feel
* Dynamic coin economy system
* Unlockable birds with rarity progression
* Shop system with lock/unlock mechanics
* Parallax background effects
* Procedural sounds generated in code
* Daily rewards and progression
* Smooth animations and gameplay polish

## Controls

* SPACE → Jump
* ESC → Return to menu
* P → Pause
* X → Restart
* Mouse → UI interaction

## Unlockable Birds
